About This Project

This Smart Home Automation & Security System was created as part of the Fall 2025 IoT: Design and Prototyping of Connected Devices course.

Project Overview

This system integrates multiple sensors, actuators, and cloud services to provide real-time monitoring, automation, and remote control capabilities.

  • DHT22 sensor for temperature & humidity monitoring
  • PIR motion sensor for security/intrusion detection
  • Three LEDs (yellow/green/red) for status indicators
  • Servo motor for door/lock control
  • Buzzer for alerts
  • Adafruit IO for cloud communication (MQTT + REST API)
  • Flask Web App using Chart.js for live & historical data visualization
  • NeonDB (PostgreSQL) for cloud-based historical data storage

System Features

  • Live sensor dashboard with real-time charts
  • Historical environmental data visualization
  • Security system control + intrusion logs
  • Remote device control (LEDs, servo, buzzer)

Architecture

The Raspberry Pi reads the sensors and controls the actuators. It publishes data to Adafruit IO and stores time-series data in a cloud PostgreSQL database (NeonDB). The Flask web application retrieves data from both the cloud and local APIs, displaying it to the user through interactive charts and controls.
